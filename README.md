# Домашнее задание по блоку 2, разделу 10

## Описание проекта
В рамках проекта выполнен анализ базы вакансий.  
Исследование направлено на изучение структуры рынка труда, особенностей вакансий, работодателей и отдельного сегмента вакансий, связанных с дата-сайенс.

## Цель работы
Провести комплексный анализ вакансий и на основе полученных данных сделать выводы:
- о распределении вакансий по регионам
- о зарплатных характеристиках
- о востребованных форматах занятости
- об активности работодателей
- об особенностях вакансий, связанных с дата-сайенс

## Состав работы
- предварительный анализ данных  
- детальный анализ вакансий  
- анализ работодателей  
- предметный анализ  
- дополнительный анализ данных  

## Краткие результаты
По итогам анализа установлено:
- рынок вакансий в подавляющем большинстве сосредоточен в крупных городах
- наиболее распространённый формат — полный рабочий день и полная занятость
- значительная часть работодателей относится к IT- и смежным направлениям
- сегмент вакансий дата-сайенс заметен, но остаётся относительно узким
- для вакансий в сфере Data Science особенно важны Python, SQL и наличие практического опыта

## Итоговый вывод
Проведённый анализ показал, что рынок вакансий в исследуемой базе достаточно крупный и структурированный. Основной спрос сосредоточен в крупных городах и ориентирован на специалистов с опытом. Направление дата-сайенс остаётся перспективным, но предъявляет высокие требования к навыкам и уровню подготовки кандидатов
